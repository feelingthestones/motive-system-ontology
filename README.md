## 🌟 这是什么？

**动机系统论（MSO）** 是一个试图统一物理学、生命科学、意识研究与哲学的新本体论框架。

我提出一个简单却深刻的核心命题：

> **能量想要流动**  
> 为了流动，能量创造了差异（对称性破缺）  
> 差异产生了：粒子、力、时空、生命、意识、文明  
> 所有复杂性都是能量寻找更好流动方式的结果

---

## 🎯 核心观点

1. **能量是唯一的存在** - 物质、信息、意识都是能量的不同形态
2. **流动是能量的本性** - 不需要外部原因，流动就是原因
3. **差异是流动的前提** - 对称性破缺创造了宇宙
4. **动-机-系-统是流动的四阶段** - 从倾向到契机到组织到稳定
